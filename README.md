# Kairo Henrique - Portfólio

Portfólio profissional desenvolvido com React, TypeScript, Tailwind CSS e Framer Motion.

## 🚀 Tecnologias

- **React 18** - Biblioteca JavaScript para construção de interfaces
- **TypeScript** - Superset JavaScript com tipagem estática
- **Vite** - Build tool moderna e rápida
- **Tailwind CSS** - Framework CSS utility-first
- **Framer Motion** - Biblioteca de animações para React

## 📦 Instalação

```bash
# Instalar dependências
npm install
```

## 🛠️ Desenvolvimento

```bash
# Iniciar servidor de desenvolvimento
npm run dev

# Acessar em http://localhost:5173
```

## 🏗️ Build

```bash
# Gerar build de produção
npm run build

# Preview do build
npm run preview
```

## 📁 Estrutura do Projeto

```
src/
├── components/
│   ├── Navbar.tsx      # Barra de navegação
│   ├── Hero.tsx        # Seção hero com apresentação
│   ├── About.tsx       # Sobre mim
│   ├── Skills.tsx      # Tecnologias e ferramentas
│   ├── Projects.tsx    # Projetos em destaque
│   ├── Contact.tsx     # Formulário de contato
│   └── Footer.tsx      # Rodapé
├── App.tsx             # Componente principal
├── main.tsx            # Entry point
└── index.css           # Estilos globais
```

## ✨ Features

- ✅ **Design moderno e responsivo** - Mobile-first, adaptável a todos os tamanhos de tela
- ✅ **Dark mode nativo** - Tema escuro elegante e confortável
- ✅ **Animações avançadas com Framer Motion:**
  - Animações ao scroll (viewport animations)
  - Stagger effects (elementos aparecem em sequência)
  - Hover states interativos e complexos
  - Gradientes dinâmicos e efeitos de brilho
  - Partículas flutuantes em background
  - Ícones e cards animados
- ✅ **Ícones de tecnologias SVG** - Logos reais de React, Vue, TypeScript, Java, Python, etc.
- ✅ **Seções interativas:**
  - Cards com lift effect e rotação
  - Tech badges com shimmer effect
  - Progress bars animadas
  - Contadores e indicadores visuais
- ✅ **Totalmente tipado com TypeScript** - Type-safe e com autocompletar
- ✅ **Performance otimizada** - Build rápido com Vite
- ✅ **SEO friendly** - Meta tags e estrutura semântica

## 📝 Personalização

Para personalizar o portfólio:

1. **Informações pessoais**: Edite os textos em cada componente
2. **Cores**: Modifique as cores em `tailwind.config.js`
3. **Projetos**: Adicione seus projetos em `src/components/Projects.tsx`
4. **Skills**: Atualize suas tecnologias em `src/components/Skills.tsx`

## 🎨 Tema de Cores

O tema utiliza uma paleta de cores customizada:

- **Primary**: Tons de azul (#0ea5e9 - #075985)
- **Dark**: Tons de cinza escuro (#020617 - #f8fafc)
